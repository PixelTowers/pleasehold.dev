**Shared setup:**
- Import `TRPCError` from `@trpc/server`
- Import `and, count, desc, eq, ilike, inArray, or, sql` from `drizzle-orm`
- Import `z` from `zod`
- Import `entries, projects` from `@pleasehold/db`
- Import `protectedProcedure, router` from `../trpc`
- Define `const entryStatusEnum = z.enum(['new', 'contacted', 'converted', 'archived'])`
- Create a reusable ownership check helper function within the file:
  ```typescript
  async function verifyProjectOwnership(db: Database, projectId: string, userId: string) {
    const project = await db.query.projects.findFirst({
      where: and(eq(projects.id, projectId), eq(projects.userId, userId)),
      columns: { id: true },
    });
    if (!project) {
      throw new TRPCError({ code: 'NOT_FOUND', message: 'Project not found' });
    }
    return project;
  }
  ```
  Import the `Database` type from `@pleasehold/db` for this helper.

**Procedure 1: `list`** (DASH-01 backend)
- Input: `{ projectId: z.string().uuid(), search: z.string().optional(), status: entryStatusEnum.optional(), page: z.number().int().min(1).default(1), pageSize: z.number().int().min(1).max(100).default(25) }`
- Verify project ownership
- Build dynamic where conditions array starting with `eq(entries.projectId, input.projectId)`
- If `input.status` provided, push `eq(entries.status, input.status)`
- If `input.search` provided, push `or(ilike(entries.email, pattern), ilike(entries.name, pattern), ilike(entries.company, pattern))!` where `pattern = '%' + input.search + '%'` (no wildcard escaping needed for v1 -- see research Pitfall 1)
- Combine conditions with `and(...conditions)`
- Use `Promise.all` to parallel-fetch: (a) paginated rows with `orderBy(desc(entries.createdAt))`, `limit(input.pageSize)`, `offset((input.page - 1) * input.pageSize)`, and (b) total count via `sql<number>'cast(count(*) as integer)'`
- Return `{ entries: rows, total, page: input.page, pageSize: input.pageSize, totalPages: Math.ceil(total / input.pageSize) }`

**Procedure 2: `getById`** (DASH-02 backend)
- Input: `{ projectId: z.string().uuid(), entryId: z.string().uuid() }`
- Verify project ownership
- Query single entry: `where: and(eq(entries.id, input.entryId), eq(entries.projectId, input.projectId))`
- If not found, throw `TRPCError({ code: 'NOT_FOUND', message: 'Entry not found' })`
- Return the entry

**Procedure 3: `stats`** (DASH-03 backend)
- Input: `{ projectId: z.string().uuid() }`
- Verify project ownership
- Use `Promise.all` for parallel queries:
  (a) Total count: `select({ total: sql<number>'cast(count(*) as integer)' }).from(entries).where(eq(entries.projectId, input.projectId))`
  (b) Status breakdown: `select({ status: entries.status, count: sql<number>'cast(count(*) as integer)' }).from(entries).where(eq(entries.projectId, input.projectId)).groupBy(entries.status)`
- Return `{ total: totalResult[0]?.total ?? 0, byStatus: Object.fromEntries(statusCounts.map(row => [row.status, row.count])) }`

**Procedure 4: `updateStatus`** (DASH-05 backend)
- Input: `{ projectId: z.string().uuid(), entryId: z.string().uuid(), status: entryStatusEnum }`
- Verify project ownership
- Update entry: `set({ status: input.status, updatedAt: new Date() }).where(and(eq(entries.id, input.entryId), eq(entries.projectId, input.projectId)))` with `.returning()`
- If no rows returned, throw `TRPCError({ code: 'NOT_FOUND', message: 'Entry not found' })`
- Return the updated entry

**Procedure 5: `bulkUpdateStatus`** (DASH-06 backend)
- Input: `{ projectId: z.string().uuid(), entryIds: z.array(z.string().uuid()).min(1).max(500), status: entryStatusEnum }`
- Verify project ownership
- Update entries: `set({ status: input.status, updatedAt: new Date() }).where(and(eq(entries.projectId, input.projectId), inArray(entries.id, input.entryIds)))` with `.returning({ id: entries.id })`
- Return `{ updatedCount: result.length }`
- SECURITY: where clause MUST include both `projectId` AND `inArray` -- prevents cross-project entry manipulation

**Procedure 6: `export`** (DASH-04 backend)
- Input: `{ projectId: z.string().uuid() }`
- Verify project ownership
- Fetch project with field config: `ctx.db.query.projects.findFirst({ where: and(eq(projects.id, input.projectId), eq(projects.userId, ctx.user.id)), with: { fieldConfig: true } })`
- Fetch ALL entries for the project (no pagination): `select().from(entries).where(eq(entries.projectId, input.projectId)).orderBy(desc(entries.createdAt))`
- Add a guard: if entries count exceeds 10,000, throw `TRPCError({ code: 'PAYLOAD_TOO_LARGE', message: 'Export limited to 10,000 entries. Contact support for larger exports.' })`
- Return `{ entries: rows, fieldConfig: { collectName: project.fieldConfig?.collectName ?? false, collectCompany: project.fieldConfig?.collectCompany ?? false, collectMessage: project.fieldConfig?.collectMessage ?? false } }`

Export the router as `export const entryRouter = router({ list, getById, stats, updateStatus, bulkUpdateStatus, export: exportProcedure })`. Note: use `exportProcedure` as the variable name internally since `export` is a reserved word, but register it as `export` in the router object.
