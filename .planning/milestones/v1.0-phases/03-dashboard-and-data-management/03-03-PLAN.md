The directory `entries/` may need to be created under `$projectId/`.

Route definition: `createFileRoute('/projects/$projectId/entries/$entryId')`.

Implementation:
1. Extract `projectId` and `entryId` from `Route.useParams()`.
2. Query entry: `trpc.entry.getById.useQuery({ projectId, entryId })`.
3. Query project: `trpc.project.getById.useQuery({ id: projectId })` — for project name in breadcrumb and field config context.
4. Status update mutation: `trpc.entry.updateStatus.useMutation({ onSuccess: () => { utils.entry.getById.invalidate({ projectId, entryId }); utils.entry.list.invalidate(); utils.entry.stats.invalidate(); } })`.
5. Get trpc utils: `const utils = trpc.useUtils()`.

Render:
- **Breadcrumb navigation**: `← Back to entries` linking to `/projects/$projectId/entries` with params.
- **Header row**: Entry email as h1 (fontSize 1.5rem, fontWeight 600), with `EntryStatusBadge` showing current status beside it.
- **Status selector**: A `<select>` dropdown with options for all 4 statuses. Value bound to current entry status. onChange calls `mutation.mutate({ projectId, entryId, status: e.target.value })`. Show saving indicator when mutation is pending. Style: padding 0.375rem 0.75rem, border 1px solid #d1d5db, borderRadius 0.375rem, fontSize 0.875rem.
- **Details section**: Rendered as a card (border 1px solid #e5e7eb, borderRadius 0.5rem, backgroundColor #fff, padding 1.5rem).
  - Display as a definition list or key-value grid (display grid, gridTemplateColumns '10rem 1fr', gap 0.75rem 1rem):
    - **Email**: entry.email
    - **Name**: entry.name or '—' if null (only show row if project collectsName, or always show since it might have data)
    - **Company**: entry.company or '—' if null
    - **Message**: entry.message or '—' if null
    - **Queue Position**: `#${entry.position}`
    - **Status**: EntryStatusBadge
    - **Submitted**: entry.createdAt formatted as locale date+time string
    - **Last Updated**: entry.updatedAt formatted as locale date+time string
  - Labels: fontWeight 500, color #6b7280, fontSize 0.875rem
  - Values: fontSize 0.875rem, color #111827
- **Metadata section** (only render if entry.metadata is not null and has keys):
  - Section heading: "Metadata" (h2, fontSize 1rem, fontWeight 600, marginTop 1.5rem)

- **Loading state**: Same centered loading pattern as project overview.
